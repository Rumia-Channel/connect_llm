# Suggested Commands
- `cargo fmt` : format the crate.
- `cargo check` : typecheck the library and binary.
- `cargo test` : run unit tests and doctests.
- `cargo run` : run the placeholder binary in `src/main.rs`.
- `git status --short` : inspect working tree changes.
- PowerShell equivalents commonly used here: `Get-ChildItem`, `Get-Content`, `rg --files`, `rg -n`.
